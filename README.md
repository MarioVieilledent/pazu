# Pazu

Pour l'instant une substitution au dictionaire Dibi qui ne marche plus (`https://dibi-dictionary.onrender.com/`)

## Utilisation

Chercher des mots Dibi / Français

Déployé en ligne : `https://pazu.alwaysdata.net/`

Liste des mots JSON (figée après site original down) : `https://pazu.alwaysdata.net/dict.json`

## Contribuer

Clone repo

```
git clone https://github.com/MarioVieilledent/pazu.git
```

Go inside repo

```
cd pazu
```

Install dependencies

```
npm i
```

Run locally

```
npm run dev
```
